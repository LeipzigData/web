Auswertung von Daten zur Corona-Pandemie

Quelle sind die von der Johns Hopkins Universität veröffentlichten Daten, die in deren github Repository veröffentlicht werden. Die Daten enthalten kumulierte Zeitreihen ab 22.01.2020 über die Zahl der Infizierten, Genesenen und Gestorbenen aufgeschlüsselt auf einzelne Länder (US-Daten sind noch kleinteiliger erfasst).

Die Daten werden für ausgewählte Länder in der Anfangsphase der Pandemie analysiert, in der mögliche exponentielle Verlaufsformen genauer extrapoliert werden.
